### Break-even 분석

**자동화로 인한 연간 절감액**:
```
Option A 인건비: $17,500/년
Option B 인건비: $5,000/년
절감액: $12,500/년
```

**추가 투자 회수 기간**:
```
추가 투자: $121,000 (Phase 3)
추가 연회비: $16,096/년 (ICAO PKD)
순 추가 비용: $121,000 + $16,096 = $137,096

연간 절감: $12,500
Break-even: $137,096 ÷ $12,500 = 11년
```

**결론**: 소규모 시스템에서는 **ICAO 회원 가입이 비용 효율적이지 않음**

### 의사결정 트리

```
1. 연간 수동 작업 시간 추정
   ├─ < 500시간 → Phase 1-2 (ICAO 공개)
   ├─ 500-1,000시간 → Phase 1-2 + 양자 협정
   ├─ 1,000-2,000시간 → Phase 1-2-3 검토 (ROI 계산 필요)
   └─ > 2,000시간 → Phase 1-2-3 (ICAO 회원 필수)

2. 법적/규제 요구사항
   ├─ 개발/파일럿 → ICAO 공개 OK
   ├─ 상용 서비스 → ICAO 공개 + 양자 협정
   └─ 국경 관리 → ICAO 회원 필수

3. 가용 예산
   ├─ < $200k → Phase 1-2
   ├─ $200-300k → Phase 1-2 + 양자 협정 강화
```
