구축 시 고려사항

1
ICAO PKD 자동 연계 수신

CSR 발급 후 LDAP V3/REST API를 통한 인증서/CRL 자동 다운로드 구현. 현재 수동 업로드만 지원. 2026년 하반기 ICAO 인증서 발급 후 연동 예정

2
Oracle Enterprise Edition 전환

현재 XE 21c (SGA 1GB 한계, PROCESSES 150) → EE (8 core, Named 200 users) 전환 필요. 코드는 Multi-DBMS 지원 완료

3
운영환경 설정 UI 확장

기관별/단말별/심사유형(유인/무인)별 인증 적용 설정 화면. 현재 API Client별 설정만 지원

4
DL 전용 조회 페이지

발급상 결함정보(Deviation List) 목록/상세 조회 별도 페이지. 현재 업로드 상세에서만 DL 구조 표시

5
서버 보안 SW 연동

SecuveTOS(서버보안), Sycros(모니터링), Chakra Max(DB접근제어)와의 감사 로그 연계/접근제어 호환 검증

6
PA 검증 데이터 장기 보관 정책

SAN 스토리지 150TiB 활용 계획 수립. SOD/DG 바이너리 데이터 보존 기간, 아카이브 정책, 백업 주기 정의
