ICAO Local PKD - 비용 분석 보고서

Version: 2.37.0 Date: 2026-03-18 Status: Planning

Note (v2.37.0): 초기 비용 분석 이후 다음 기능이 추가되었으나, 비용 수치는 당초 추정치를 그대로 유지합니다:

  • CSR Management (ICAO PKD Certificate Signing Request 생성/관리)
  • API Client Management (외부 클라이언트 API Key 인증/권한/Rate Limiting)
  • Pending DSC Approval (PA 검증 자동 등록 → 관리자 승인 워크플로우)
  • EAC 인증서 실험적 지원 (BSI TR-03110 CVC 파서/검증)

개요

본 문서는 ICAO Local PKD 시스템 구축 및 운영에 필요한 비용을 단계별로 분석합니다. 데이터 소스 선택, 자동화 수준, 시스템 규모에 따라 비용이 크게 달라지므로, 여러 시나리오를 비교하여 최적의 투자 전략을 제시합니다.

데이터 소스별 비용

ICAO 공식 PKD

1. 공개 다운로드 (무료)

비용: $0

제약사항:

  • 수동 다운로드만 가능
  • 운영 시스템 용도 부적합 (ToS 위반)
  • 자동화 불가
  • ICAO 기술 지원 없음

적합 용도:

  • 개발 및 테스트
  • 파일럿 프로젝트
  • 학술 연구

Break-even 분석

자동화로 인한 연간 절감액:

Option A 인건비: $17,500/년
Option B 인건비: $5,000/년
절감액: $12,500/년

추가 투자 회수 기간:

추가 투자: $121,000 (Phase 3)
추가 연회비: $16,096/년 (ICAO PKD)
순 추가 비용: $121,000 + $16,096 = $137,096

연간 절감: $12,500
Break-even: $137,096 ÷ $12,500 = 11년

결론: 소규모 시스템에서는 ICAO 회원 가입이 비용 효율적이지 않음

   ├─ < 500시간 → Phase 1-2 (ICAO 공개)
   ├─ 500-1,000시간 → Phase 1-2 + 양자 협정
   ├─ 1,000-2,000시간 → Phase 1-2-3 검토 (ROI 계산 필요)
   └─ > 2,000시간 → Phase 1-2-3 (ICAO 회원 필수)

   ├─ 개발/파일럿 → ICAO 공개 OK
   ├─ 상용 서비스 → ICAO 공개 + 양자 협정
   └─ 국경 관리 → ICAO 회원 필수

3. 가용 예산
   ├─ < $200k → Phase 1-2
   ├─ $200-300k → Phase 1-2 + 양자 협정 강화
   └─ > $300k → Phase 1-2-3 검토
